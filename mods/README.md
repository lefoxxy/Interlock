# Mod Distribution

Use this directory only for CurseForge-facing notes or generated export metadata that belongs in source control.

Do not commit downloaded mod jars.

Preferred workflow:

1. Maintain the playable profile in the CurseForge app.
2. Keep configs, scripts, quests, and datapack content in this repository.
3. Generate the CurseForge export when you want to test install flow or prepare a release.
