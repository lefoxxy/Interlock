# Quest Structure

Interlock quests should teach factory thinking, not demand repetitive crafting.

The quest line is organized around operational milestones:

- establish reliable power
- build visible logistics before digital control
- automate meaningful production chains with multiple mods

Quests should avoid:

- raw item grind
- passive waiting
- duplicate objectives that restate recipe gates

## Tree Overview

### 1. Factory Charter

Purpose:
Introduce the pack's expectations, progression logic, and cross-mod identity.

Key outcomes:

- understand that logistics matter early
- understand that AE2 is a later convenience layer
- understand that IE starts power, Mekanism scales processing, and IF specializes automation

### 2. Stable Current

Purpose:
Make the player establish an early power grid with Immersive Engineering before broader machine access.

Key outcomes:

- reliable IE generation
- LV storage or buffering
- first powered workshop

### 3. Routed Materials

Purpose:
Teach chunk-spanning movement and item routing with physical transport before AE2.

Key outcomes:

- Pipez-based transport over meaningful distance
- machine input and output routing
- non-manual movement between work areas

### 4. Process Discipline

Purpose:
Transition into Mekanism as the primary processing backbone.

Key outcomes:

- core Mekanism machine line online
- ore processing automated with at least two mods participating
- increased power demand becomes visible and intentional

### 5. Grid Expansion

Purpose:
Position Powah as the mid-game power bridge and reward a stronger factory grid.

Key outcomes:

- Powah infrastructure running alongside IE or Mekanism
- stable power under sustained machine load
- first real energy storage milestone

### 6. Automation Milestone

Purpose:
Unlock Industrial Foregoing only after the player already owns power, processing, and transport.

Key outcomes:

- plastic chain completed
- first IF automation machine deployed
- a farm, fluid, or mob workflow improved rather than replaced wholesale

### 7. Systems Control

Purpose:
Make AE2 feel earned after physical logistics has already solved early transport.

Key outcomes:

- ME Controller crafted
- drive and first storage tier online
- AE2 supplements Pipez rather than replacing the early game

## Quest Authoring Rules

- Prefer completion by machine placement, multiblock completion, or producing a proof item once.
- Prefer "show the system works" over "craft 64 of X".
- Where possible, require outputs from automation rather than handcrafting every intermediate.
- Use optional quests for optimization tips and sidegrades.
- Keep rewards small and instructional; recipes and progression are the main reward.
