# Industrial Role Map

This pack uses a strict industrial split.

## Primary Roles

- `Immersive Engineering`: early game machines and power
- `Mekanism`: primary processing and mid-game machines
- `Industrial Foregoing`: late automation for farms, mobs, and fluids

## Overlap Summary

- IF generators undermine IE's role as the first real power system.
- IF laser/resource machines bypass the intended IE -> Mekanism factory arc.
- IF processing blocks blur the line between Mekanism as the main machine spine and IF as an automation specialist.
- IF farm, mob, and fluid machines are appropriate for the pack, but only after Mekanism and IE infrastructure already exists.
